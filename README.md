## 💫 Floorpプロジェクトとは？
<strong>「Floorpプロジェクトは、プライバシーを追求し、匿名性とウェブの公開性のバランスを追求するプロジェクトです。」</strong>

Firefox ベースながらも全く違うエクスペリエンスをお届けします。Floorp はカスタマイズ性を優先したブラウザーです。
